# 运维管理系统

数字政府统一运维管理平台，实现"监、管、控、析、安、服"一体化管理。

## 功能模块

### 1. 服务门户
- **快速提单**: 支持外部运维人员快速提交工单
- **服务目录**: 可视化服务分类，便于用户选择
- **工单查询**: 实时查看工单状态和处理进度
- **知识库检索**: 快速查找相关解决方案

### 2. 工单管理
- **工单列表**: 支持多维度筛选和搜索
- **流转历史**: 完整记录工单处理过程
- **数据统计**: 图表化展示工单量和状态分布
- **自动分配**: 系统自动分配工单给处理人员
- **高级搜索**: 支持任意字段作为搜索项

### 3. 资产台账管理
- **资产台账**: 统一管理IT资产信息
- **客户管理**: 管理委办局等客户信息
- **项目管理**: 管理各委办局项目信息
- **关联引用**: 资产信息可在工单中直接引用

### 4. 自定义表单
- **表单设计器**: 支持自定义表单字段
- **字段类型**: 文本、数字、日期、下拉框、复选框、单选框、附件等
- **字段校验**: 支持必填、格式校验等规则
- **表单模板**: 可保存为模板供复用

### 5. 服务管理
- **服务目录管理**: 统一管理IT服务场景
- **流程配置**: 自定义变更、事件、请求、问题等流程
- **服务类型**: 支持四种服务管理类型

### 6. 例行工作
- **定时任务**: 支持每日/每周/每月自动执行
- **自动提单**: 按预设任务自动创建工单
- **任务调度**: 后台自动调用流程进行流转

### 7. 知识库
- **文章管理**: 支持多种文章类型
- **版本管理**: 记录文章修改历史
- **标签分类**: 自定义文章标签便于检索
- **批量导入**: 支持批量导入知识文章

### 8. 监控告警
- **告警接收**: 接收监控系统推送的告警
- **工单关联**: 告警自动创建工单并关联
- **闭环管理**: 从告警到恢复的完整闭环

### 9. AI智能问答
- **知识库应答**: AI机器人基于知识库回答问题
- **运维助手**: 提供故障排查建议和操作指导

### 10. 系统设置
- **用户管理**: 管理用户账户和权限
- **角色权限**: 支持管理员、内部人员、外部人员三种角色
- **双因素认证**: 支持短信或动态口令双因素认证
- **系统日志**: 记录所有操作日志

## 技术架构

### 前端技术栈
- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript 5
- **UI组件**: shadcn/ui (基于 Radix UI)
- **样式**: Tailwind CSS 4
- **图表**: Recharts
- **状态管理**: React Hooks

### 后端技术栈
- **运行时**: Node.js
- **API**: Next.js API Routes
- **数据库**: PostgreSQL
- **ORM**: Drizzle ORM
- **AI集成**: Coze LLM SDK

- **knowledge_articles**: 知识库文章表
- **service_catalog**: 服务目录表
- **workflows**: 流程定义表
- **form_templates**: 表单模板表
- **scheduled_tasks**: 例行任务表
- **monitor_alerts**: 监控告警表
- **audit_logs**: 操作日志表



## 访问系统

开发环境默认运行在 `http://localhost:5000`

### 主要页面路由
- `/` - 仪表板
- `/login` - 登录页面
- `/tickets` - 工单管理
- `/assets` - 资产台账
- `/knowledge` - 知识库
- `/services/catalog` - 服务目录
- `/scheduled-tasks` - 例行工作
- `/monitoring` - 监控告警
- `/portal/quick-submit` - 快速提单
- `/settings/users` - 用户管理

## API 接口

### 工单管理
- `GET /api/tickets` - 获取工单列表
- `POST /api/tickets` - 创建新工单

### AI 问答
- `POST /api/ai/chat` - AI 智能问答（流式响应）
